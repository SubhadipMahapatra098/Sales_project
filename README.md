## Introduction 

This project showcases a simple Sales Dashboard built using *Power Bi* to analyze key metrics like **sales, profit, and quantity**. It provides **interactive visuals and filters** for quick insights and better decision-making.

## ✨ Skills Showcased

- **⚙️ Data Cleaning & Transformation:** Cleaned and structured raw sales data using Power Query by handling missing values and correcting data types.

- **📊 DAX Calculations:** Created measures like Total Revenue, Total Profit, Total Cost, Units Sold, and Profit Margin.

- **📈 Data Visualization:** Built Bar Charts, Line Charts, Pie Charts, and Scatter Plots to analyze performance across different dimensions.

- **📅 Time Series Analysis:** Analyzed order trends over time to identify patterns and fluctuations.

- **🌍 Regional Analysis:** Compared sales across countries and regions to identify top-performing markets.

- **📌 KPI Indicators:** Designed cards to highlight key metrics like Revenue, Profit, Cost, and Margin.

- **🎨 Dashboard Design:** Created a visually appealing and user-friendly layout with consistent color themes and alignment.

- **🧠 Interactive Reporting:**
  - **Slicers:** Region, Country, Order Priority, Sales Channel, Item Type  
  - **Drill-Through:** Detailed analysis from overview page  
  - **Navigation Button:** Smooth transition between pages
